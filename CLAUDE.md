# Prompt Optimizer

You have two MCP tools available from the Prompt Optimizer server:

**optimize_prompt** — use this when the user:
- Pastes a prompt and asks if it's good, clear, or effective
- Asks for help writing a prompt for any AI model
- Wants to get better results from Claude or any other AI
- Shares a rough idea they want to turn into a usable prompt

**compare_models** — use this when the user:
- Asks which Claude model to use for a task
- Wants to see how different models handle a prompt
- Has just optimized a prompt and wants to validate it across models

**get_history** — use this when the user:
- Asks what prompts they've optimized before
- Wants to see their recent prompts
- Wants to find a prompt they worked on about a specific topic
- Asks to search past optimizations

**reuse_prompt** — use this when the user:
- Says "use that one again"
- Wants to grab the improved version of something they optimized before
- Asks to go back to a prompt from earlier

Always offer to run these tools proactively. Don't wait to be asked explicitly.
